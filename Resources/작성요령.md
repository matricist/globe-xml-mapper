# Globe XML Mapper 작성요령

## 공통 안내
- Excel에서 파일을 열면 Control Panel이 자동으로 표시됩니다.
- Control Panel에서 [+][-] 버튼으로 항목을 추가/삭제할 수 있습니다.
- 작성 완료 후 메인 화면의 [XML 변환하기]로 XML을 생성합니다.
- Enum 값은 GIR 코드(예: GIR401)로 입력합니다.
- ISO 국가코드는 2자리(예: KR, JP, US), 통화코드는 3자리(예: KRW, USD)입니다.
- 복수 기재 가능 항목은 쉼표로 구분합니다 (예: KR, JP, US).
